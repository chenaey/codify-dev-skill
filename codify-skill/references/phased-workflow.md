# 复杂设计流程

**禁止一次性生成所有代码。必须拆分组件逐个实现。**

---

## 流程

```
规划 → 逐个实现 → 组装
```

### 1. 规划

获取设计数据后，创建 `structure.md`：

```markdown
| 组件 | 节点 ID | 状态 |
|------|---------|------|
| Header | 377:68199 | ⬜ |
| ProductCard | 500:028698 | ⬜ |
```

状态：⬜ 待实现 → ✅ 已完成

### 2. 逐个实现

循环直到所有组件 ✅：
1. 选择下一个 ⬜ 组件
2. 生成代码（遵循 [codegen-rules.md](codegen-rules.md)）
3. 下载资源
4. 更新状态为 ✅

### 3. 组装

所有组件 ✅ 后，创建页面入口文件组装组件。
